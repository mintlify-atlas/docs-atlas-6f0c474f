## What is Oat UI?

Oat is an ultra-lightweight HTML + CSS, semantic UI component library with zero dependencies. No framework, build, or dev complexity. Just include the tiny CSS and JS files and you are good to go building decent looking web applications with most commonly needed components and elements.

Semantic tags and attributes are styled contextually out of the box without classes, forcing best practices, and reducing markup class pollution. A few dynamic components are WebComponents and use minimal JavaScript.

## Why Oat UI?

Oat was created after the unending frustration with the over-engineered bloat, complexity, and dependency-hell of pretty much every Javascript UI library and framework out there. The goal is a simple, minimal, vanilla, standards-based UI library that you can use in your projects for the long term without having to worry about Javascript ecosystem churn.

The look and feel are influenced by the shadcn aesthetic, with automatic light/dark theme support and modern design patterns.
